##### 栏目添加

按岗位区分

```
只能查询出自己的岗位

添加时，默认添加自己的岗位
```

不按岗位区分

```
可以查询出所有岗位

添加时，可以指定所有岗位
```

##### 人员设置

按岗位区分

```
只能查询自己的岗位

添加时，默认添加自己的岗位
```

不按岗位区分

```
可以查询所有的岗位

添加时，岗位为空
```

##### 分配模板

按岗位区分

```
分配分数：按自己的岗位分配

栏目：按自己的岗位查询生成

人员：按自己的岗位查询生成
```

不按岗位区分

```
分配分数：科室总分

栏目：所有栏目

人员：所有人员

人员的岗位和栏目的岗位不匹配，不能分配分数。即为0。
```
