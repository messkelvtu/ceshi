### shiro框架

#### 表设计

###### 用户表

```txt
用户ID
```

###### 角色表

```txt
角色ID、父角色ID
```

###### 用户角色表

```txt
ID、用户ID、角色ID
```

###### 权限表

```txt
权限ID、权限标识、父权限ID
```

###### 角色权限表

```txt
ID、角色ID、权限ID
```

##### 查询用户是否有该权限

```txt
用户ID---根据用户ID查--》用户角色表---根据角色ID查--》角色权限表
```

##### 添加用户权限

   实际就是添加该角色的权限，即添加一条角色权限表的数据

```txt
1、前端展示权限表的树结构。
2、根据用户ID查到用户角色表。
3、再根据角色ID和前端提供的权限ID，在角色权限表添加一条记录。
```
